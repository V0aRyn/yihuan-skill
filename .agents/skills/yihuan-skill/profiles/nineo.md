# nineo 玩家档案

本文件属于玩家档案层，用于没有指定档案时的默认规划偏好。用户本次输入优先级高于本档案。

## 用户倾向

- 付费：大小月卡党，默认大月卡 68 元档；接受高性价比礼包和首充双倍。
- 资源态度：均衡达成目标，优先保证角色、关键武器和版本活动。
- 方斯态度：版本末保留 100-200 万方斯；买房版本和联动版本按本次目标单独判断。

## 目标优先级

1. 角色：默认追求全图鉴，明确不喜欢的角色可跳过。
2. 武器：每版本规划 1-2 把关键武器。
3. 版本活动：联动和限定活动倾向满档。
4. 好感度：坚持日常投入，这部分视为最低保障。
5. 都市目标：买房、载具、收藏按用户输入计入。
6. 安全垫：版本末保留方斯余量。

## 活动与商店

- 联动/限定活动：倾向满档或完成用户指定档位。
- 猎人交易所：每版本常规购买环石、三重钥匙和永恒的螺音。
