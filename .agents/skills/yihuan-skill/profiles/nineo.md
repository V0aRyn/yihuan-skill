# nineo 玩家档案

用于没有指定档案时的默认规划偏好。用户本次输入优先级高于本档案。

## 付费与礼包偏好

```text
大小月卡党
愿意购买礼包
双倍首充优先留给皮肤，礼包不够时再用于兜底
```

## 默认解释

- 版本收入默认按 `大小月卡党` 档位计算。
- 礼包建议可以列入高性价比礼包，但不要默认把所有礼包都视为已购买。
- 双倍首充默认不主动动用；只有目标缺口明显、礼包仍不足或用户明确允许时才作为兜底。
- 用户输入的付费档位、礼包预算、双倍首充策略和底仓要求覆盖本档案。

## 好感度培养

默认每日策略：

1. 主培养角色：2 个 400 好感度礼物 + 免费云朵道具 + 免费约会。
2. 其他角色：剩余 7 次送 200 好感度礼物。
3. 主培养角色每日获得：1400 好感度。
4. 主培养角色每日付费成本：30,000-40,000 方斯。
5. 如果剩余送礼次数全部给其他角色，中级礼物按 3,000 方斯/次计算。

主培养角色满级估算：

```text
56000 / 1400 = 40 days
付费礼物成本：约 120 万-160 万方斯
```
